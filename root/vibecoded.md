
This document lists AI assisted development tools used during the implementation of the project and relevant configurations

## AI Assisted Development Tools Used

ChatGPT OpenAI was used for debugging backend API issues resolving integration problems and assisting with frontend backend communication

Gemini Google was used for architecture clarification and troubleshooting environment configuration issues

## Supporting Configurations

Backend Environment Variables

MONGODB_URI configured for MongoDB Atlas connection  
GROQ_API_KEY configured for AI model access  
FRONTEND_URL configured for frontend communication setup  

Frontend Environment Variables

VITE_BACKEND_URL configured for backend API communication  

Deployment Setup

Frontend deployed on Vercel using React and Vite build  
Backend deployed on Vercel as a Node.js service  
Repository structured as a monorepo with separate frontend and backend directories

AI tools were used as development assistance for debugging configuration and integration support.
